# Installation

Let there be @ember packages:

```sh
  git clone https://github.com/emberjs/ember.js vendor/ember.js
  git checkout v3.6.0
  cd ../..
  sudo sh scripts/init.sh
  sudo node scripts/npm-link-packages.js
  npm install # NOTE: VERY IMPORTANT
  parcel index.html
```
